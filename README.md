# Healthcare Dashboard using Power BI

## Project Overview
This project analyzes healthcare data using Power BI to provide insights into patient admissions, hospital performance, treatment costs, and department-wise analysis.

## Features
- Patient Analysis
- Department Performance
- Treatment Cost Analysis
- Interactive Filters
- KPI Cards
- Charts and Visualizations

## Tools Used
- Power BI Desktop
- Microsoft Excel / CSV
- Power Query
- DAX
